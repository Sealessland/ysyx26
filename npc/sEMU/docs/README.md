# sEMU (C++23 Modern Emulator Environment)

欢迎来到 sEMU 的开发文档主页。此文档由 Doxygen 与 `doxygen-awesome-css` 自动生成。

## 目录结构
- **`include/sEMU`**: 暴露给外部的标准化 C++ 头文件
- **`src/`**: 核心实现代码
- **`tests/`**: 单元测试
- **`docs/`**: 设计文档及 Doxygen 配置

## 环境
本项目基于 **C++23**，并使用 CMake 结合 Makefile 的现代化管理方式。您可以浏览左侧的菜单来查看类和文件的详细接口说明。

## 快速使用
```cpp
#include "sEMU/container.hpp"
// 愉快地进行开发
```
